Web Application Development

course code:ISM 1131

Course level:Year One,Semester One

Credit Unit:4

Course Description

This course introduces students to Web technologies-HTML,CSS,JavaScript; CSS frameworks;JavaScript frameworks. Students will learn to establish modern web pages using HTML and CSS framework. The course also introduces students to full srtack web application programming using JavaScript.

Course Objectives

  1. Describe web technologies(HTML,CSS & JavaScript), CSS frameworks, and JavaScript frameworks.
  2. Implement a simple but modern web page with HTML and CSS.
  3. Build simple but functional web application using JavaScript.

Learning Outcomes

  • Use Web technologies(HTML,CSS & JavaScript),CSS frameworks, and JavaScript frameworks.
  • Design and create a simple but functional web page using HTML and CSS frameworks.
  • Compose and build a simple gfull stack web application using JavaScript framework.
